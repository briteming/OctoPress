# Markdown Guide for OctoPress

This guide will help you format your posts beautifully using Markdown.

## Basic Formatting

### Headers

```markdown
# H1 Header
## H2 Header
### H3 Header
```

### Text Styling

- **Bold**: `**bold text**`
- *Italic*: `*italic text*`
- ~~Strikethrough~~: `~~strikethrough~~`

### Lists

Unordered list:
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2

Ordered list:
1. First item
2. Second item
3. Third item
